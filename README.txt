Packages
-----------

algo  
  The algorithms needed by the project.

chengTxn  
  The evaluation framework simulating multiple clients.

graph  
  A simple and basic graph implementation.

kvstore  
  A simple in-memory Key-Value store implementation, called cheng-KV.

kvstore/exception
  Exceptions defined on top of cheng-KV.

kv_interfaces  
  The client library for cheng-KV.

kv_interfaces/instrument  
  An instrumented cheng-KV library.

verifier  
  Different implementations of Cobra's verifier.

verifier/txngraph  
  The graph data structures used by Cobra's verifier, including
  precedence graph, write-dependency graph, etc.
